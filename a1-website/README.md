A1 Website
=================

Project Title: Women In Tech Website
Description
This project is a website dedicated to 'Women in Tech', featuring a responsive design with several web pages. These pages include a form for submitting credit card details, typography examples, and more. All project files are located in the `public/`./ directory.

Getting Started

1. Clone the Repository
    To clone the repository and view the project on your local machine, execute the following command:
    git clone https://gitlab.lnu.se/1dv528/student/gy222at/a1-website
    cd a1-website


2) Install Dependencies.
    Before running the project, make sure you install all required dependencies:
    npm install



3) . Running the Linters
    To ensure code quality and adherence to styling guidelines, run the linters with the following command:
    npm run lint


4) Starting the Web Server
    npm run http-server
    Once the web server is running, open your web browser and navigate to http://localhost:[port-number]. Replace [port-number] with the actual port number used by your web server.
