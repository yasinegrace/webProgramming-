# Project Title: Personal Web Desktop (PWD)

## Description
The Personal Web Desktop (PWD) application is a web-based desktop environment that allows users to interact with various applications such as a 
# 1 Memory Game, 
    Click the Memory Game icon to open the game window.
    Select your game size: 2x2, 4x4, or 2x4. The game board will appear with your chosen size
    To play a different size or start a new game, simply move the current game window aside and click the Memory Game icon again.
    Choose your new game size and start playing. You can play multiple games at once by repeating these steps.
# 2  Chat
    click on the chat icon and then you will get a prompt asking you to enter your username.
    you can also change the name by clicking on change userName button
    you can also select the channel (general, TechTalk or Random)
# 3  Task Manager. 
    this application is a to-do list, allowing users to create, edit, and delete tasks, and toggle them between complete and incomplete states.
